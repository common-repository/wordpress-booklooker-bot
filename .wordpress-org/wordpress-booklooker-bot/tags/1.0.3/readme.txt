== Description ==

Ein einfaches Wordpress-Plugin um Deine Booklookerangebote auch auf Deinem Blog listen zu können. Das Weltbild-Marktplatz System läuft auf Booklooker-Basis, deshalb sollte das Plugin auch mit Weltbild-Marktplatz funtionieren. Dieses Plugin darf jeder kostenlos benutzen, verändern und auch kostenlos weiterreichen. Bevor Ihr allerdings das Plugin
installiert, solltet ihr eine Sicherung Eures Blogs machen. Gewährleistung kann ich nicht anbieten - also eigenes Risiko. 

**Features**

* Einfache Installation.
* Einstellungen speicherbar.
* In jedes Template mit dem Shortcode einfügbar

== Installation ==

Nutze dafür einfach dein Dashboard

**Installation via Wordpress**

1. Geht ins Menü <strong>'Plugins' -> 'Install'</strong> und suche nach <strong>'WordPress Booklooker-bot'.</strong>
2. <strong>Klickt</strong> hier auf <strong>'install'.</strong>
3. Geht zum Plugin WordPress-Booklooker-bot und <strong>aktiviert es.</strong>
4. Hat das geklappt, <strong>klickt</strong> in der Navigation <strong>auf Plugins-Untermenüpunkt: Booklooker-bot Otpionen</strong>
5. Dort <strong>tragt</strong> ihr eure <strong>Booklooker UID,</strong> die <strong>Medien-Type/Kategorie</strong> ein und <strong>ändert die Überschrift/Einleitung</strong> (optional) nach eurem Wunsch.
6. Danach den <strong>Shortcode</strong> (zu finden auf der Optionseite) <strong>in eine Seite</strong> (z. B.: Neue Seite mit Standardtemplate
   erstellen) <strong>oder in einen Artikel</strong> an einer Stelle deiner Wahl <strong>einfügen und veröffentlichen</strong>

**Manuelle Installation**

1. Die Zip-Datei <strong>Wordpress-booklooker-bot runterladen</strong>
2. Die Datei <strong>auf dem eigenen Rechner entpacken</strong> (Ergebnis: 2 Dateien in einem Ordner mit dem Namen: booklooker_bot).
3. Den Ordner <strong>booklooker_bot ins Plugin-Verzeichnis des WordPressblogs hochladen.</strong>
4. Meldet euch in der Admin des Blogs an und <strong>navigiert zu Plugins (anzeigen).</strong>
5. Geht dort <strong>zum Plugin WordPress-Booklooker-bot und aktiviert es.</strong>
6. Hat das geklappt, <strong>klickt</strong> in der Navigation <strong>auf Plugins-Untermenüpunkt: Booklooker-bot Otpionen</strong>
7. Dort <strong>tragt</strong> ihr eure <strong>Booklooker UID,</strong> die <strong>Medien-Type/Kategorie</strong> ein und <strong>ändert die Überschrift/Einleitung</strong> (optional) nach eurem Wunsch.
8. Danach den <strong>Shortcode</strong> (zu finden auf der Optionseite) <strong>in eine Seite</strong> (z. B.: Neue Seite mit Standardtemplate
   erstellen) <strong>oder in einen Artikel</strong> an einer Stelle deiner Wahl <strong>einfügen und veröffentlichen</strong>
