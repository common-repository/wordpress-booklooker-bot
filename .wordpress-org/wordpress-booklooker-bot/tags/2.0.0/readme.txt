== Description ==

Ein einfaches Wordpress-Plugin um Deine Booklookerangebote auch auf Deinem Blog listen zu können. Das Weltbild-Marktplatz System läuft auf Booklooker-Basis, deshalb sollte das Plugin auch mit Weltbild-Marktplatz funtionieren. Es kann jetzt bis zu 6 unterschiedliche Seiten mit maximal 150 Deiner Booklookerartikel, Ober- und Unterkategorien darstellen. Auch die Darstellung mehrerer Booklooker-Accounts möglich. Dieses Plugin darf jeder kostenlos benutzen, verändern und auch kostenlos weiterreichen. Bevor Du das Plugin
installierst, solltest Du eine Sicherung Deines Blogs machen. Gewährleistung kann ich nicht anbieten - also eigenes Risiko. 

**Features**

* Einfache Installation.
* 6 Optionseiten für unterschiedliche ebenso viele Seiten/Artikel.
* Optionseiten mit Hilfefunktion ausgestattet.
* Automatisch nach Optionsauswahl erstellte Shortcodes.
* Shortcode in Seiten oder Artikel einfach einfügbar.
* Durch ein stylesheet gut an Dein Blog anpassbar.
* Bis zu 900 Artikel, auf insgesamt 6 Seiten anzeigbar.

== Installation ==

Nutze dafür einfach Dein Dashboard.

**Installation via Wordpress**

1. Gehe ins Menü und navigiere zu <strong>'Plugins' -> 'Install'</strong> und suche nach <strong>'WordPress Booklooker-bot'.</strong>
2. <strong>Klicke</strong> hier auf <strong>'install'.</strong>
3. Gehe zum Plugin WordPress-Booklooker-bot und <strong>aktiviere es.</strong>
4. Hat das geklappt, <strong>klicke</strong> in der Navigation <strong>auf den Plugins-Untermenüpunkt: Booklooker-bot Otpionen</strong>
5. Dort <strong>trage</strong> ihr Deine <strong>Booklooker UID,</strong> die <strong>Medien-Type/Kategorie</strong> ein und <strong>ändert die Überschrift/Einleitung</strong> (optional) nach Deinem Wunsch.
6. Danach den <strong>Shortcode</strong> (zu finden auf der Optionseite) <strong>in eine Seite</strong> (z. B.: Neue Seite mit Standardtemplate
   erstellen) <strong>oder in einen Artikel</strong> an einer Stelle Deiner Wahl <strong>einfügen und veröffentlichen.</strong>

**Manuelle Installation**

1. Die Zip-Datei <strong>Wordpress-booklooker-bot runterladen.</strong>
2. Die Datei <strong>auf dem eigenen Rechner entpacken.</strong>
3. Den Ordner <strong>booklooker_bot ins Plugin-Verzeichnis Deines WordPressblogs hochladen.</strong>
4. Melde Dich in der Admin des Blogs an und <strong>navigiere zu Plugins (anzeigen).</strong>
5. Gehe dort <strong>zum Plugin WordPress-Booklooker-bot und aktiviere es.</strong>
6. Hat das geklappt, <strong>klicke</strong> in der Navigation <strong>auf Plugins-Untermenüpunkt: Booklooker-bot Otpionen</strong>
7. Dort <strong>trage</strong> Deine <strong>Booklooker UID,</strong> die <strong>Medien-Type/Kategorie</strong> ein und <strong>ändere die Überschrift/Einleitung</strong> (optional) nach Deinem Wunsch.
8. Danach den <strong>Shortcode</strong> (zu finden auf der Optionseite) <strong>in eine Seite</strong> (z. B.: Neue Seite mit Standardtemplate
   erstellen) <strong>oder in einen Artikel</strong> an einer Stelle Deiner Wahl <strong>einfügen und veröffentlichen.</strong>

== Changelog ==

= 2.0.0 =
* Änderung des Datenabrufs auf eine API-Version
* Es können jetzt 150 Artikel pro Seite angezeigt werden. Insgesamt sind das 900 Artikel.
* Der Zugriff funktioniert auf die neue Weise schneller und sicherer.
* Bessere, leichtere Anpassung an Dein Layout durch ein vorhandenes Booklooker-bot-Plugin-Stylesheet. Wenn Du im Stylesheet Anpassungen vornimmst, speichere die Datei style_wpbb.css nach den abgeschlossenen Änderungen per FTP auf Deinem Rechner, sonst werden Deine Anpassungen bei einem Update überschrieben.
* Das komplette Stylesheet ist mit Kommentaren verfasst, die Dir die Einstellungen leicht machen sollten.
* Ab jetzt gibt es eine Feinanpassung an die verschiedenen Produktkategorien
* Ersatzbilder erscheinen, wenn Du keine Produktbilder beim Einstellen Deiner Artikel bei Booklooker verlinkt hast.
* Defekte Bilderlinks werden nicht zur Nichtanzeige gebracht. Stattdessen wird der, aus dem Titel und dem Autor, generierte Alt-Tag anstatt des Bildes angezeigt (unschön).
* Die Ersatzbilder book_standard.png, music_standard.png, film_standard.png und game_standard.png können natürlich durch eigene mit gleichem Namen ersetzt werden.

= 1.1.1 =
* In der Version 1.1.0 gab es einen Bug, der den Upload von Mediendateien in Artikel oder Seiten verhinderte. Das sollte mit dieser Version 1.1.1 behoben sein. Sorry an alle Betroffenen.
