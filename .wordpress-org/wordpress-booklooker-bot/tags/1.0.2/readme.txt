== Description ==

Ein einfaches Wordpress-Plugin um Deine Booklookerangebote auch auf Deinem Blog listen zu können. Das Weltbild-Marktplatz System läuft auf Booklooker-Basis, deshalb sollte das Plugin auch mit Weltbild-Marktplatz funtionieren. Dieses Plugin darf jeder kostenlos benutzen, verändern und auch kostenlos weiterreichen. Bevor Ihr allerdings das Plugin
installiert, solltet ihr eine Sicherung Eures Blogs machen. Gewährleistung kann ich nicht anbieten - also eigenes Risiko. 

**Features**

* Einfache Installation.
* Einstellungen speicherbar.
* In jedes Template mit dem Shortcode einfügbar

== Installation ==

Nutze dafür einfach dein Dashboard

**Installation via Wordpress**

1. Geht ins Menü 'Plugins' -> 'Install' und suche nach 'WordPress Booklooker-bot'
2. Klickt hier auf 'install'
3. Geht zum Plugin WordPress-Booklooker-bot und aktiviert es
4. Hat das geklappt, klickt in der Navigation auf Plugins-Untermenüpunkt: Booklooker-bot Otpionen
5. Dort tragt ihr eure Booklooker UID, die Medien-Type/Kategorie ein und ändert die Überschrift/Einleitung (optional) nach eurem Wunsch.
6. Danach den Shortcode (zu finden auf der Optionseite) in eine Seite (z. B.: Neue Seite mit Standardtemplate
   erstellen) oder in einen Artikel an einer Stelle deiner Wahl einfügen und veröffentlichen

**Manuelle Installation**

1. Die Datei booklooker_bot.zip runterladen
2. Die Datei auf dem eigenen Rechner entpacken (Ergebnis: 2 Dateien in einem Ordner mit dem Namen: booklooker_bot)
3. Den Ordner booklooker_bot ins Plugin-Verzeichnis des WordPressblogs hochladen
4. Meldet euch in der Admin des Blogs an und navigiert zu Plugins (anzeigen)
5. Geht dort zum Plugin WordPress-Booklooker-bot und aktiviert es
6. Hat das geklappt, klickt in der Navigation auf Plugins-Untermenüpunkt: Booklooker-bot Otpionen
7. Dort tragt ihr eure Booklooker UID, die Medien-Type/Kategorie ein und ändert die Überschrift/Einleitung (optional) nach eurem Wunsch.
8. Danach den Shortcode (zu finden auf der Optionseite) in eine Seite (z. B.: Neue Seite mit Standardtemplate
   erstellen) oder in einen Artikel an einer Stelle deiner Wahl einfügen und veröffentlichen
